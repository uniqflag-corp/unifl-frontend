## 📄 페이지별 구성

### 1. 메인 페이지 (`index.html`)

| 섹션 | 설명 |
|------|------|
| Hero | 파티클 배경 + 슬로건 + CTA 2종 (무료 상담 / 견적 확인) |
| Why UNIFL | 유니플 강점 6가지 카드 |
| Services | 서비스 4종 (홈페이지, 쇼핑몰, 랜딩페이지, 맞춤형 솔루션) |
| Portfolio | 제작 사례 6개 (더미) + 전체보기 링크 |
| Process | 5단계 타임라인 (상담→디자인→개발→검수→유지보수) |
| Stats | 신뢰 지표 카운터 (150+ 프로젝트 / 98% 만족도 / 8년 경력 / 100% 납기) |
| FAQ | 아코디언 8개 항목 |
| CTA | 무료 상담 유도 + 전화번호 |

---

### 2. 회사소개 (`about.html`)
- 비전 & 미션
- 핵심 가치 3가지 (고객 중심, 혁신, 신뢰)
- 오시는 길 (지도 영역 + 연락처 정보)

---

### 3. 서비스 (`service/homepage.html`)
- 제작 유형 4종 (랜딩페이지, 기업 홈페이지, 쇼핑몰, 맞춤형 솔루션) + 가격
- 제작 특징 6개 (반응형, SEO, 소스코드 납품, 유지보수, 빠른 속도, 1:1 전담)

---

### 4. 포트폴리오 (`portfolio.html`)
- 필터 탭 (전체, 기업 홈페이지, 쇼핑몰, 랜딩페이지, 예약 시스템)
- 더미 프로젝트 10개 (그라데이션 카드)

---

### 5. 자동 견적 계산기 (`estimate.html`) ⭐

4단계 위자드 방식의 실시간 견적 계산기:

| 단계 | 내용 | 입력 방식 |
|------|------|----------|
| ① 사이트 유형 | 랜딩(80만~), 홈페이지(150만~), 쇼핑몰(250만~), 맞춤형(400만~) | 카드 선택 |
| ② 페이지 & 디자인 | 서브 페이지(15만/개), 게시판(10만/개), 디자인 수준 | ± 수량 조절 + 카드 |
| ③ 기능 선택 | 회원, 결제, 콘텐츠, 예약 등 12개 기능 | 체크박스 다중 선택 |
| ④ 부가 서비스 | 관리자, SEO, 다국어, 알림, 유지보수 등 10개 | 체크박스 + 카드 |

**핵심 기능:**
- 실시간 Sticky 사이드바 합계 표시
- VAT 별도 표기
- "이 견적으로 상담받기" CTA → 문의 페이지 연결
